CSS Box Model Deep Dive

The CSS box includes margin, border, padding, and content.

1. Content - The actual content (text, images, etc.)
2. Padding - Space inside the border, around the content
3. Border - A line around the padding
4. Margin - Space outside the border (doesn't count toward element width)

Visual Box Model

PADDING: 20px
